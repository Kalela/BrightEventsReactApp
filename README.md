## Project Information
Bright Events is a event management app that allows users to create, manage and view other users events. You can find the apps backend [here](https://github.com/Kalela/BrightEvents).
## Install

1. Clone the repo:
```
$ git clone [URL]
```
2. Install dependencies.
```
$ npm install
```
3. Run the app:
```
$ npm start
```
4. Open [http://localhost:8000](http://localhost:8000) to view it in the browser.
## Running tests

```
$ npm test
```
